🕰️ ChronoChoice

A minimalist, data-driven watch discovery platform designed to help users make clear, unbiased decisions when choosing a watch.

🚀 Overview

ChronoChoice is a front-end web application that allows users to:

Discover watches through filters and search
Compare multiple watches side-by-side
Explore detailed specifications
Access only authentic, trusted timepieces

This project focuses on clarity over clutter, eliminating unnecessary noise commonly found in watch marketplaces.

🎯 Objectives
Provide a simple and intuitive interface for watch selection
Ensure no fake or replica products are listed
Maintain complete neutrality (no sponsorship bias)
Highlight watches based on quality, usability, and reliability
✨ Features
🔍 Discovery System
Search by name or description
Filter by:
Movement (Quartz, Automatic, Mechanical, Solar)
Budget
Sort by price (Low → High / High → Low)
⚖️ Compare Tool
Compare up to 3 watches simultaneously
Highlights best price automatically
One-click Clear Compare option
🖼️ Visual UI
Watch cards with image previews
Smooth hover animations
Scroll-based fade-in effects
📄 Detailed Watch Pages
Individual pages for each watch
Image carousel for product viewing
Complete specifications
Direct purchase links to official sources
🧠 UX Enhancements
Empty state handling
Active navigation highlighting
Responsive layout
